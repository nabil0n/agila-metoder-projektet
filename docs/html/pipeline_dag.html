pipeline_dag module¶

pipeline_dag.tesing_functions()¶

This function defines a DAG (Directed Acyclic Graph) for a pipeline process. It consists of several tasks that are executed sequentially. Tasks: - run_newsfeed: Downloads blogs from an RSS feed and extracts articles. - summarize_articles: Summarizes the extracted articles. - send_webhook: Sends the summarized articles to Discord. - clear_bucket: Clears a bucket. Execution Flow: 1. run_newsfeed task is executed. 2. summarize_articles task is executed. 3. send_webhook task is executed. 4. clear_bucket task is executed. Returns: - The current timestamp when each task is completed.
